# Quản lý đào tạo

Web quản lý học viên, khóa học và đăng ký. Dữ liệu lưu trên trình duyệt (localStorage).

## Chạy project

Cần Node.js 18+.

```bash
npm install
npm run dev
```

Mở `http://localhost:5173` (hoặc URL Vite hiển thị trong terminal).

## Trang chính

- `/` — Tổng quan
- `/student` — Học viên
- `/course` — Khóa học
